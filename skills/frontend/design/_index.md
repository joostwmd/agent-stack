# Design Skills

UI and design quality skills for creating, refining, and auditing interfaces.

## When to use

| Task | Load |
|------|------|
| Adapt across devices/contexts | adapt/SKILL.md |
| Add motion, micro-interactions | animate/SKILL.md |
| Audit quality (a11y, perf, theming) | audit/SKILL.md |
| Amplify safe/boring designs | bolder/SKILL.md |
| Improve UX copy, labels, errors | clarify/SKILL.md |
| Add strategic color | colorize/SKILL.md |
| Evaluate design effectiveness | critique/SKILL.md |
| Add joy, personality | delight/SKILL.md |
| Strip to essence, simplify | distill/SKILL.md |
| Extract components, tokens | extract/SKILL.md |
| Build distinctive UIs from scratch | frontend-design/SKILL.md |
| Improve resilience, edge cases | harden/SKILL.md |
| Align with design system | normalize/SKILL.md |
| Design onboarding, empty states | onboard/SKILL.md |
| Optimize performance | optimize/SKILL.md |
| Final polish pass | polish/SKILL.md |
| Tone down intensity | quieter/SKILL.md |
| One-time design context setup | teach-impeccable/SKILL.md |

**Load only the skill relevant to the current task.** Do not load all skills.
